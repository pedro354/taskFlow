# 🚀 TaskFlow

Sistema Fullstack para gerenciamento de tarefas e subtarefas, desenvolvido com React, TypeScript, Node.js e PostgreSQL.

O objetivo do projeto é permitir a organização de atividades através de uma interface moderna e intuitiva, oferecendo controle completo sobre tarefas, subtarefas e progresso das atividades.

---

## 📋 Funcionalidades

* Criar tarefas
* Editar tarefas
* Excluir tarefas
* Criar subtarefas
* Atualizar status das tarefas
* Organização por categorias
* Interface responsiva
* Integração entre Frontend e Backend
* Persistência de dados em banco PostgreSQL

## ⚙️ Instalação

### Clone o repositório

```bash
git clone https://github.com/seu-usuario/taskflow.git
```

### Entre na pasta do projeto

```bash
cd taskflow
```

### Instale as dependências

Frontend:

```bash
npm install
```

Backend:

```bash
npm install
```
