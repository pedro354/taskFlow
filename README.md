# TaskFlow

Sistema Fullstack de gerenciamento de tarefas e subtarefas desenvolvido com React, TypeScript, Node.js e PostgreSQL.

---

## 📌 Sobre o projeto

O **TaskFlow** nasceu como um projeto de estudos focado em:

* React com TypeScript
* Gerenciamento de estado com Context API
* Backend com Node.js e Express
* Integração Fullstack
* PostgreSQL
* Estruturação de componentes
* Animações com Framer Motion
* Arquitetura e organização de projeto

O objetivo principal foi construir uma aplicação com aparência moderna e funcionamento próximo de sistemas reais de produtividade.

# ⚙️ Como executar o projeto

## Clone o repositório

```bash
git clone https://github.com/seu-usuario/taskflow.git
```

---

# 💻 Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend disponível em:

```bash
http://localhost:5173
```

---

# 🛠️ Backend

```bash
cd backend
npm install
node server.js
```

Backend disponível em:

```bash
http://localhost:4000
```

---

# 🗄️ Banco de dados

O projeto utiliza PostgreSQL.

Configure suas variáveis de ambiente:

```env
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=sua_senha
DB_NAME=taskflow
```
